# 📱 Mobile Number Lookup Tool — Termux Edition

A Python-based CLI tool for Termux (Android) that looks up mobile number details via a REST API and displays formatted results.

## ✨ Features

- 🔍 Look up mobile number details (name, father name, aadhaar, address, etc.)
- 🎨 Clean, color-formatted CLI output
- 🔄 Continuous loop — query multiple numbers without restarting
- ✅ 10-digit validation
- ❌ Graceful error handling
- 📦 Zero configuration — clone and run

## ⚙️ Requirements

- Termux (Android)
- Internet connection

## 🚀 Installation

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/number_lookup.git

# Enter the directory
cd number_lookup

# Run the installer
bash install.sh

# Launch the tool
python main.py
